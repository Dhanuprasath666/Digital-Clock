# 🕒 Digital Clock

A simple Digital Clock built using **HTML, CSS, and JavaScript**. This project displays the current time in real-time with hours, minutes, seconds, milliseconds, and AM/PM.

## 🚀 Features

- Real-time digital clock
- Displays:
  - Hours
  - Minutes
  - Seconds
  - Milliseconds
  - AM/PM format
- Updates continuously using JavaScript

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📁 Project Structure

```
Digital Clock/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `html.html` in your browser.
3. The digital clock will start automatically.

## 📚 What I Learned

- Working with the JavaScript `Date` object
- Updating the UI using the DOM
- Using `setInterval()` for real-time updates
- Formatting time values
- Basic HTML, CSS, and JavaScript integration
